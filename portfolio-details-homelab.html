Home Lab — Docker Containers on Ubuntu

A personal homelab running on an Ubuntu machine, using Docker containers managed through Portainer. The stack covers media streaming, photo management, a services dashboard, file access, and local AI/LLM experimentation.

  • Jellyfin — Self-hosted media server for movies, TV, and music.
  • Immich — Self-hosted photo and video backup (Google Photos alternative).
  • Homarr — Customisable home dashboard to launch all services at a glance.
  • FileBrowser — Lightweight web-based file manager over the Docker volumes.
  • OpenAI / Open WebUI — Local LLM interface (Ollama + Open WebUI) for AI experimentation.

All stacks are defined as Docker Compose files and deployed / maintained through the Portainer UI, with no vendor lock-in and full data ownership.
